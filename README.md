# 🌟 BE-KIND Interactive Educational Game

An interactive educational game developed using **C++** and **OpenGL** that encourages players to make positive decisions through engaging real-life scenarios.

## 📖 About
This project was developed as part of a Computer Science course. The game presents interactive real-life situations where players make choices that encourage kindness, responsibility, and positive behavior through engaging gameplay.

## ✨ Features
- Interactive gameplay
- Multiple animated scenes
- Decision-based scenarios
- Sound effects
- 2D graphics using OpenGL

## 🛠️ Technologies
- C++
- OpenGL
- GLUT / FreeGLUT
- GLEW
